Docker Plex

The purpose of this project is to install Plex Media Server easily on a server (PC/Mac/Linux/RaspberryPi) through Docker technology

Easy install with a single line of code

docker run -d --restart=always --name plex -v /mnt/usbdrive:/media --net=host jaymoulin/plex

and then access to Plex via this url : http://your_machine_ip:32400

Google Music Manager - Python Daemon

Daemon to upload/download Google Music Library to Google Music

Easy install with a simple lines of code

pip install google-music-manager-auth

pip install google-music-manager-uploader

pip install google-music-manager-downloader

Download easy usage
  1. Auth : google‑music‑auth
  2. Download : cd /your/download/folder && google‑music‑download
Upload easy usage
  1. Auth : google‑music‑auth
  2. Upload : cd /your/upload/folder && google‑music‑upload

SSHTron - Docker image

SSHTron is a multiplayer lightcycle game that runs through SSH. Just run the command below and you'll be playing in seconds:

Easy install with a single line of code (Windows/Linux/Mac/RaspberryPi...)

Twidge (Twitter CLI) - Docker image

This image allows you to Twitter in CLI (Command line interface) easily thanks to Docker.

Easy install with a single line of code (Windows/Linux/Mac/RaspberryPi...)

docker run --rm -ti -v /path/to/your/.twidgerc:/root/.twidgerc jaymoulin/twidge setup

Use this command to know want you can do then

docker run --rm -ti -v /path/to/your/.twidgerc:/root/.twidgerc jaymoulin/twidge lscommands

Or this command to help

docker run --rm -ti -v /path/to/your/.twidgerc:/root/.twidgerc jaymoulin/twidge --help
