Docker Plex

This purpose of this project is to install Plex Media Server easily on a server (PC/Mac/Linux/RaspberryPi) through Docker technology

Easy install with a single line of code

docker run -d --restart=always --name plex -v /mnt/usbdrive:/media --net=host jaymoulin/plex

and then access to Plex via this url : http://your_machine_ip:32400

Docker JDownloader

This purpose of this project is to install JDownloader easily on a server (PC/Mac/Linux/RaspberryPi) through Docker technology

Easy install with a single line of code

docker run -d --restart=always -v ~/Downloads:/root/Downloads -v ~/jdownloader/cfg:/opt/JDownloader/cfg --name jdownloader jaymoulin/jdownloader

Google Music Manager - Python Daemon

Daemon to upload/download Google Music Library to Google Music

Easy install with a simple lines of code

pip install google-music-manager-auth

pip install google-music-manager-uploader

pip install google-music-manager-downloader

Download easy usage
  1. Auth : google‑music‑auth
  2. Download : cd /your/download/folder && google‑music‑download
Upload easy usage
  1. Auth : google‑music‑auth
  2. Upload : cd /your/upload/folder && google‑music‑upload
